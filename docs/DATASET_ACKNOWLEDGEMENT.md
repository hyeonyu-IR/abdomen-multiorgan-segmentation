# Dataset acknowledgement: AMOS22

This project uses the **AMOS22** abdominal multi-organ segmentation dataset, distributed via the AMOS22 Grand Challenge.

- AMOS22 Grand Challenge portal: https://amos22.grand-challenge.org/

## Recommended dataset citation

Please cite the AMOS dataset paper when using AMOS/AMOS22 in academic work:

- https://arxiv.org/abs/2206.08023
- NeurIPS Datasets and Benchmarks paper PDF: https://proceedings.neurips.cc/paper_files/paper/2022/file/ee604e1bedbd069d9fc9328b7b9584be-Paper-Datasets_and_Benchmarks.pdf

**Ji Y, Bai H, Yang J, et al.**  
*AMOS: A Large-Scale Abdominal Multi-Organ Benchmark for Versatile Medical Image Segmentation.*  
NeurIPS Datasets and Benchmarks (2022).

## Notes

- This repository does **not** include AMOS22 imaging data.
- Users must ensure they follow the dataset usage terms and any applicable institutional/data-governance requirements.
